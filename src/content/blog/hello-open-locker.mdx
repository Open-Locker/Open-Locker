Open Locker ist ein Open Source Projekt für **Software und Hardwarebaupläne** von öffentlichen Schließfächern zum Lagern und/oder Teilen von Gegenständen.

## Warum wir das bauen

In Hameln-Pyrmont werden öffentliche Schließfächer bereits genutzt, um Gegenstände wie Laptops oder VR-Headsets zu verleihen. Open Locker soll die User Experience verbessern und das System leichter anpassbar machen.

## Wie du mitmachen kannst

- Wöchentliches Treffen jeden Dienstag um **19:30 CET / 18:30 UTC**
- Austausch auf Discord
- Issues und Pull Requests auf GitHub
